## Dataset Description
The dataset used in this lab is a tabular housing dataset obtained from an open-source platform.
It contains 17,000 records and 9 numerical features, including location information, population data, and income levels.
The dataset is suitable for machine learning tasks.

## Machine Learning Problem
This is a supervised machine learning problem.
The task is a regression problem where the goal is to predict the median house value based on the given features.
